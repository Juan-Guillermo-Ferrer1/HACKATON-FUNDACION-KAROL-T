# 📋 Guía de Evaluación - Hackatón Fundación ProMiTierra

## Criterios de Evaluación

| Criterio | Peso | Descripción |
|----------|------|-------------|
| **UX/UI** | 25% | Facilidad de uso, diseño visual, experiencia del usuario |
| **Creatividad** | 25% | Originalidad, innovación, solución única al problema |
| **Utilidad** | 25% | Valor real, aplicabilidad, impacto potencial |
| **Calidad Técnica** | 25% | Limpieza del código, arquitectura, buenas prácticas |

## Requisitos Obligatorios

- [ ] Proyecto plenamente funcional
- [ ] README con instrucciones claras de ejecución
- [ ] Repositorio público accesible
- [ ] Sin contenido ofensivo o inapropiado
- [ ] Proyecto nuevo o sin usuarios reales previos

## Stack Tecnológico

No hay restricciones. Los participantes pueden usar cualquier tecnología que consideren apropiada para su solución.
